如何免登录从 Flaticon 下载 SVG 格式的图标
✨小透明・宸✨
2025-12-06 22:39:47

Flaticon 是我常用的一个提供免费商用图标素材下载的网站。Flaticon 的图标质量很高，用于制作 PPT、示意图或者其他设计用途都特别适合。在未登录的情况下，Flaticon 的图标支持保存为清晰度最大为 512px 的 PNG 位图，但是保存 SVG、EPS 矢量图和 PSD 格式则需要开通会员。为了追求清晰度我会倾向于使用 SVG 矢量图，于是这个限制就非常难受了。

还好 Flaticon 留了一个空子可以钻：在图标的页面上有个“Edit icon”的按钮，登录后可以对图标进行改色等简单的编辑（不需要开通会员），在打开编辑器的过程中会请求 SVG 格式的图标，这时通过 F12 打开浏览器开发者工具或者使用各种提取网页上 SVG 图标的扩展就可以把图标保存下来。

很多相关的教程（例如这个教程：独家揭秘！教你从 Flaticon 和 Iconfinder 批量下载免费图标！ - 知乎）都提到了这一点。不过使用这个方法还是需要注册登录，而我连找个临时邮箱注册都懒得动手……

先说结论：我在我的服务器上部署了一个服务，可以直接获取 Flaticon 的图标。

首先还是得注册一个账号。

登录之后会得到一些 Cookie，其中比较重要的是 .flaticon.com 下的 FI_REFRESH_TOKEN 和 FI_TOKEN 这两项。后者显然与登录状态有关，实际上是一个 JWT，解码出 payload 之后可知每次发放的 FI_TOKEN 有效期是一小时；前者根据名字判断应该是获取新的 FI_TOKEN 的。

根据测试，只携带 FI_REFRESH_TOKEN 而不携带 FI_TOKEN（或者使用过期的 FI_TOKEN）向 https://www.flaticon.com/ 发送 GET 请求（需要一个浏览器的 User-Agent），响应里会发放新的 FI_TOKEN。FI_REFRESH_TOKEN 应该是长期有效的，因此只要配置好 FI_REFRESH_TOKEN 就可以随时获取有效的 FI_TOKEN。

F12 打开浏览器开发者工具的“网络”查看请求，可以知道编辑器请求 SVG 图标的 API 是 https://www.flaticon.com/editor/icon/svg/{iconId}?type=standard。Flaticon 收录的图标分为四种类型：Icons、Stickers、Interface Icons、Animated Icons，根据这里可以知道对应的 type 分别是 standard、sticker、uicon、animated-icon。图标 ID 存在且 type 正确的话这个 API 会返回一个形如 https://cdn-icons.flaticon.com/svg/***/***.svg?token=exp=***~hmac=***" 的 URL 指向 SVG 图标（否则返回 null），显然这个 URL 必须通过这个 API 获取，否则无法解决过期时间和签名问题。

请求这个 API 需要附带 FI_REFRESH_TOKEN 和 FI_TOKEN 的 Cookie 以及一个浏览器的 User-Agent，否则会直接返回 404。

获取 SVG 图标的请求则不需要设定这些东西。比较遗憾的是通过这种方法获取的 Animated Icons 的 SVG 也是静态的，没有动画效果。
