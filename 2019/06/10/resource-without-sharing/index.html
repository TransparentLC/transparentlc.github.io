“为什么要做这个页面？”

点进去一看，嗯……“免费领取方式”？“按提示操作”？

“为什么不使用百度网盘？”

在下载资源前，小透明还是要做一些说明( ｡･_･｡)

现在一提到“发资源”，大家第一时间想到的应该都是百度网盘吧？

百度网盘凭借人均 2TB 的容量、方便的“分享”功能、当然最重要的是有钱可烧，在数年前的“网盘大战”中成为了最后的优胜者，今天的中文互联网上到处都是通过百度网盘分享的资源。

（“废话”里面提及的营销号也是百度网盘的忠实用户，甚至还将速盘之类的第三方客户端用来充当自己为“大会员”们“特别准备”的“网盘高速下载器”←_←）

但百度网盘在通过（早期的）高质量的服务吸引了绝大多数用户后，转而通过 10KB/s、“账号黑名单”这类严重降低用户体验的手段逼迫用户开通会员，作为商业行为可以理解，却给用户造成了极大不便。

所以小透明不会把这里的资源上传到百度网盘，而是上传到 Catbox 资源已转移至微软的 OneDrive 网盘！可以用 IDM 等多线程下载工具或浏览器直接下载了～( ‘-ωก̀ )

整理资源时用到的一些小工具

  • FFmpeg

    • 世界上最好的视频音频转码工具( ｀д´ )
    • 对于视频教程类的资源，小透明会用 ffmpeg -i input.mp4 -c:v libx264 -tune stillimage -crf 28 -c:a aac -b:a 48k output.mp4 二压。
    • 因为这些文件的视频部分一般是使用软件录屏直接保存，为了做到实时编码而舍弃了压缩比，还有不少进一步压缩的空间，更不用说某些录屏软件还在使用 AVI、WMV 之类的封装格式，里面是压缩率比 H.264 不知道低到哪里去了的各种古代视频编码……音频部分一般是人声录音，也不需要很高的码率。
    • 经过二压后的视频文件，大小能减少到原来的一半甚至四分之一(ゝ∀･)

  • SpaceCapture

    • 相当好用的长截图工具～
    • 这里的“预览图”都是使用它生成的ヾ(´∀｀。ヾ)

  • MozJPEG-GUI

    • 小透明使用 C# 和 WPF 自己制作的使用 MozJPEG 压图的 GUI 程序ヾ(•ω•`。)
    • 这样小透明每次需要压缩预览图的时候就不用打开 Adobe Photoshop 了～

  • ShareX

    • 截图录屏软件，不过小透明一般还是习惯使用 Windows 自带的截图工具（Windows+R 然后 snippingtool），或者直接用键盘上的截图键（っ╹◡╹)ﾉ
    • 这个神奇的软件还有编辑截图和上传截图到图床的功能……呐呐呐，作为一个截图工具，有这样的功能确实很方便～
    • 但是它还做了上传文件，生成短链接之类的功能？！
    • 总之小透明就把它当成上传文件到 Catbox 以及上传图片到马云家的服务器的工具来用，按照 API 文档配置好上传参数就可以愉快地使用了(っ’ω’)っ
    • 这软件甚至在 Steam 上架了……不用等 G 胖降价，Steam 版当然也是免费的啦！

  • PowerShell

    • Windows 自带的脚本工具，cmd 的威力加强版，是小透明经常用的东西٩( ‘ω’ )و
    • 使用 foreach ($file in Get-ChildItem) { ... } 批量操作文件（重命名、压制视频、生成压缩包、计算 MD5）都很方便！٩(ˊᗜˋ*)و

  • OneDrive for Business 直链生成工具

    • Onedrive 5TB 的白嫖教程到处都是，配合这个就可以生成直链了，除了下载还可以用来当图床用～
    • 不开源

总结了一下才发现上面的这些工具基本上都是开源软件，开源就是好～

SpaceCapture 只是免费软件但并没有开源，官方网站也无法访问了，但是有人把软件本体放到了 GitHub 上|ω•`)
